What is javascript?

Javascript is a scripting language which is enables you to make a retreive content dynamically. Javascript gives opportunity of sort out with lots of complex features. Javascript can handle hmtl and css both files. It's very lightweight and highly used on web pages. Javascript runs inside the web browsers or server.

Difference between Local storage vs Session storage

Local Storage: Local storage stores data but does not delete. For example, when you closed your web browser all the data has been stored in local storage, then after logging back you will get data. That means local storage does not have any expiration time or date.
Session Storage: Session Storage does not store data rather delete after every session. Each time user closes the web browser all the data from session storage is deleted. That means it's usable for a single time.
